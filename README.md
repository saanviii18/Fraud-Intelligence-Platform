# 💳 Fraud Intelligence Platform

An end-to-end machine learning system for real-time credit card fraud detection using XGBoost, FastAPI, and Streamlit.

## Features

* Fraud Detection
* Fraud Probability Estimation
* Risk Classification
* FastAPI REST API
* Streamlit Dashboard

## Tech Stack

* Python
* XGBoost
* FastAPI
* Streamlit
* Pandas
* NumPy

## Architecture

Dataset
↓
Data Preprocessing
↓
XGBoost Model
↓
fraud_model.pkl
↓
FastAPI API
↓
Streamlit Dashboard
↓
Fraud Prediction

## Model Performance

* Precision: 94%
* Recall: 81%
* F1 Score: 87%
